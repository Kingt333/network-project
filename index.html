OSI Model & TCP/IP

Understanding network communication through layered models

Layer 7 - Application

Supports end-user processes like HTTP, HTTPS, FTP, SMTP.

Layer 6 - Presentation

Handles data formatting, encryption, compression, translation.

Layer 5 - Session

Manages sessions, connections, synchronization between apps.

Layer 4 - Transport

Provides reliable delivery (TCP/UDP), ports, flow control.

Layer 3 - Network

Routes packets using IP, ICMP, handles addressing.

Layer 2 - Data Link

Transfers frames, handles MAC addresses, switches.

Layer 1 - Physical

Deals with cables, signals, hubs, raw bit transmission.

LAN Small area, high speed, low cost
WAN Large area, lower speed, higher cost
MAN City-wide coverage, medium speed
WLAN Wireless LAN, flexible but less secure
